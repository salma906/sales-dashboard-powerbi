# 📊 Sales Data Analysis Dashboard (Power BI)

## 📌 Overview
This project presents an interactive Power BI dashboard designed to analyze sales data and identify key trends and insights to support data-driven decision-making.

## 🛠️ Tools Used
- Power BI  
- Power Query  

## 📈 Key Features
- Interactive dashboard for sales analysis  
- Visualization of key metrics (revenue, sales trends)  
- Data cleaning and transformation using Power Query  

## 📊 Insights
- Identified sales trends and performance patterns  
- Highlighted key metrics to support business decisions
